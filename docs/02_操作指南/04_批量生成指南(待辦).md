# 04_批量生成指南 (Batch Mask Generation)

此文件提供了如何使用 `src/tools/hybrid_map_gen.py` 腳本進行批量幾何遮罩生成 (Batch Mask Generation) 的指南，並詳細說明了其可配置的新功能。

## 1. 概述

`hybrid_map_gen.py` 是一個強大的工具，能夠根據 `master_config.yaml` 中的設定，自動生成多種複雜的幾何遮罩。這些遮罩將用於後續的 LBM 模擬，為 AI 代理模型的訓練提供多樣化的數據集。腳本會自動根據 `physics_control.rho_in_list` 的長度或預設數量來生成批量的遮罩。

## 2. 配置 (master_config.yaml)

所有批量生成行為和幾何特徵的定義都集中在 `master_config.yaml` 文件的 `map_generator` 區塊中。以下是關鍵參數及其功能：

```yaml
map_generator:
  domain:
    height: 1024              # 計算域高度（像素）
    width: 4096               # 計算域寬度（像素）

  pinball:                      # 彈珠台區塊配置
    shape: ["circle", "square", "triangle"] # 形狀選擇：圓形、正方形、三角形
    center_x_ratio: 0.12      # 中心 X 位置（相對於 domain 寬度比例）
    center_y_ratio: 0.5       # 中心 Y 位置（相對於 domain 高度比例）
    size_ratio: [0.03, 0.07]      # 形狀大小（相對於 domain 短邊比例）
    spacing_factor: [1.8, 3.5]      # 多顆物體之間的間距倍率
    rotation_angle: [-45, 45]     # 方形/三角形旋轉角度（度）
    triangle_orientation: ["vertex_left", "edge_left"] # 三角形朝向：尖端朝左或邊面朝左

  tube_bank:                    # 管束區塊配置
    shape: ["circle", "square", "triangle"] # 形狀選擇
    layout: ["staggered", "aligned"]  # 排列方式：交錯或對齊
    start_x_ratio: 0.30       # 起始 X 位置比例
    end_x_ratio: 0.55         # 結束 X 位置比例
    size_ratio: [0.03, 0.09]      # 形狀大小
    num_rows: [3, 6]            # 列數
    num_cols: [4, 7]             # 行數
    jitter_amount: [10, 50]       # 隨機擾動幅度（像素）
    rotation_angle: [-15, 15]     # 方形/三角形旋轉角度（度）
    triangle_orientation: ["vertex_left", "edge_left"] # 三角形朝向

  step_urban:                   # 階梯城市區塊配置 (此區塊維持矩形)
    step_start_ratio: 0.55
    # ... 其他參數 ...

  validation:
    boundary_buffer: 10      # 邊界緩衝距離，用於合法性檢查

  output:
    invert_values: True      # 是否反轉輸出 mask 的數值定義 (0=obstacle, 1=fluid)
```

**重點說明:**
-   對於 `shape`、`layout`、`rotation_angle` 等參數，您可以提供一個單一值，或一個列表讓腳本在每次生成時隨機選擇。
-   `size_ratio` 和 `jitter_amount` 這些範圍參數，腳本會從其定義的區間內隨機取樣。

## 3. 執行批量生成

1.  **調整 `master_config.yaml`**: 根據上述說明，設定您想要的幾何特徵和生成數量。
    -   `settings.project_name`: 定義專案名稱。
    -   `physics_control.rho_in_list`: 這個列表的長度將影響生成的遮罩數量（如果列表長度大於 1，則至少生成 20 張，或取列表長度）。

2.  **執行腳本**: 運行以下命令來啟動批量遮罩生成。

    ```bash
    python src/tools/hybrid_map_gen.py
    ```

## 4. 輸出

生成的遮罩會自動儲存到 `SimCases/{project_name}/masks/` 目錄中。檔名格式為 `L{CharacteristicLength}_{ID}.png`，其中 `CharacteristicLength` 為該遮罩的特徵長度，`ID` 為自動生成的序號。

例如：
```text
SimCases/
└── Hyper-1/                # project_name
    └── masks/            # 自動生成的遮罩存放於此
        ├── L120_0000.png
        ├── L100_0001.png
        ├── L150_0002.png  # 可能包含方形、三角形等
        └── ...
```

同時，`map_gen_config.json` (這是 `map_generator` 區塊的完整複製) 也會被儲存在同一個 `masks` 資料夾中，方便追溯本次批量生成的配置細節。
